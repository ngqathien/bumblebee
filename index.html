How to Buy

You can buy Bumblebee through this link

Create MetaMask wallet

Create a MetaMask Wallet using either a desktop computer or an iOS/Android mobile device. That will allow you to buy, sell, send, and receive Bumblebee Tokens.

Send ETH to your wallet

You can buy Ethereum (ETH) directly on MetaMask or transfer it to your MetaMask Wallet from exchanges like Coinbase, Binance, etc. Make sure to use the ERC-20 network when transfering ETH.

Visit Bumblebee Exchange Page

As of now, you can purchase Bumblebee through UniSwap, Bumblebee’s official decentralized exchange where you can swap any ERC-20 Token.

Swap ETH for Bumblebee

You can start swapping as soon as you have ETH available! Press ‘Select a token’ and enter the token address or search for it on the tokens list.
